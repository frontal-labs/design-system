# @frontal/shared

Frontal's shared utilities and helpers package.

## Overview

This package provides shared utilities, helpers, and common functions used across the Frontal design system packages.

## Installation

This package is part of the Frontal monorepo and is installed automatically as a workspace dependency.

```bash
pnpm install @frontal/shared
```

## Usage

```tsx
import { /* shared utilities */ } from '@frontal/shared';
```

## Development

```bash
# Build the package
pnpm build

# Run tests
pnpm test

# Lint
pnpm lint
```
