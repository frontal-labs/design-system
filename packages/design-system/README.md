# @frontal/design-system

Frontal's core design system package.

## Overview

This package provides the foundational design system components, tokens, and utilities that form the basis of the Frontal platform's UI.

## Installation

This package is part of the Frontal monorepo and is installed automatically as a workspace dependency.

```bash
pnpm install @frontal/design-system
```

## Usage

```tsx
import { /* design system components */ } from '@frontal/design-system';
```

## Development

```bash
# Build the package
pnpm build

# Run tests
pnpm test

# Lint
pnpm lint
```
