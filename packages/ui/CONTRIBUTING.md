# Contributing to @frontal/ui

This package is part of the Frontal Design System monorepo. Please refer to the [root CONTRIBUTING.md](../../CONTRIBUTING.md) for general contribution guidelines.

## Package-Specific Notes

- This package contains foundational UI components (Box, Container, Grid, Flex, Stack, etc.)
- All components should be fully typed with TypeScript
- Follow the existing component patterns and prop naming conventions
