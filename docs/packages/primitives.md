# @frontal/primitives

Low-level primitive components for the Frontal Design System.

## Overview

`@frontal/primitives` provides foundational primitive components that serve as building blocks for higher-level UI components.

## Installation

This package is part of the Frontal monorepo and is available as a workspace dependency.

## Usage

Primitives are typically used internally by other packages, but can be imported directly:

```tsx
import { Primitive } from "@frontal/primitives";
```

## Purpose

Primitives serve as:

- **Building blocks** for UI components
- **Base components** with minimal styling
- **Reusable foundations** for custom components
- **Accessibility foundations** with proper semantics

## When to Use

Use primitives when:

- Building custom components
- Need base functionality without styling
- Creating component libraries
- Building on top of the design system

## Dependencies

None (standalone package)

## Peer Dependencies

- `react` >= 19.2.0
