# Contributing to @frontal/blocks

This package is part of the Frontal Design System monorepo. Please refer to the [root CONTRIBUTING.md](../../CONTRIBUTING.md) for general contribution guidelines.

## Package-Specific Notes

- This package contains UI blocks and components
- All blocks follow the registry pattern defined in `src/components/_registry.ts`
- When adding new blocks, ensure they are properly registered in the registry
- Follow the existing block structure and naming conventions
