# Contributing to @frontal/typeface

This package is part of the Frontal Design System monorepo. Please refer to the [root CONTRIBUTING.md](../../CONTRIBUTING.md) for general contribution guidelines.

## Package-Specific Notes

- This package contains font files and typeface utilities
- Font files are stored in the `fonts/` directory
- When adding new fonts, ensure proper fallback handling is implemented
- Follow the existing font loading patterns
