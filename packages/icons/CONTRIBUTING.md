# Contributing to @frontal/icons

This package is part of the Frontal Design System monorepo. Please refer to the [root CONTRIBUTING.md](../../CONTRIBUTING.md) for general contribution guidelines.

## Package-Specific Notes

- This package contains icon components generated from SVG files
- Icons are organized by category in the `icons/` directory
- When adding new icons, use the icon generation script to ensure consistency
- Follow the existing icon naming conventions
